Основные функции проекта
1. validate_card_number(card_number: str) -> bool
Описание: Эта функция проверяет, является ли введенный номер карты валидным.
Логика:
Проверяет, что длина номера карты составляет 16 символов.
Использует регулярное выражение для проверки, состоит ли номер только из цифр.
Возвращаемое значение: True, если номер карты валиден; False в противном случае.
2. validate_account_number(account_number: str) -> bool
Описание: Проверяет, валиден ли номер банковского счета.
Логика:
Убеждается, что длина номера счета составляет 20 символов.
Использует регулярное выражение для подтверждения, что номер состоит только из цифр.
Возвращаемое значение: True для валидного номера счета; False в противном случае.
3. get_mask_card_number(card_number: str) -> str
Описание: Возвращает замаскированный номер карты, чтобы скрыть часть информации.
Логика: Показывает первые четыре цифры, затем две цифры, затем заменяет остальные на звёздочки, оставляя видимыми последние четыре цифры.
Возвращаемое значение: Замаскированный номер карты (например, 7000 79** **** 6361).
4. get_mask_account(account_number: str) -> str
Описание: Возвращает замаскированный номер банковского счета.
Логика: Показывает только последние четыре цифры счета, остальные заменяются на звёздочки.
Возвращаемое значение: Замаскированный номер счета (например, **4305).
Функции для работы с транзакциями
5. filter_by_state(data, state='EXECUTED')
Описание: Фильтрует список транзакций по заданному состоянию (по умолчанию EXECUTED).
Логика: Использует списковое включение для возврата только тех транзакций, которые соответствуют указанному состоянию.
Возвращаемое значение: Список транзакций с заданным состоянием.
6. sort_by_date(data, descending=True)
Описание: Сортирует список словарей по дате транзакции.
Логика: Использует sorted() с функцией datetime.strptime() для преобразования строковых дат в объекты datetime.
Возвращаемое значение: Отсортированный список словарей по дате.
Функции для маскирования и форматирования даты
7. mask_account_card(card_account: str) -> str
Описание: Принимает строку с наименованием типа счета или карты и маскирует номер.
Логика: Проверяет, содержит ли строка слово "Счет", и применяет соответствующую функцию маскирования. Собирает все алфавитные части строки и добавляет замаскированный номер.
Возвращаемое значение: Строка с названием типа и маскированным номером.
8. get_date(date: str) -> str
Описание: Преобразует дату из формата Г.Г.Д. в формат Д.М.Г..
Логика: Извлекает день, месяц и год из строки даты и форматирует их в нужный порядок.
Возвращаемое значение: Дата в формате Д.М.Г. (например 11.03.2024).
